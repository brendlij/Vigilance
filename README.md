## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) or [Bun](https://bun.sh/)
- A modern web browser

### Installation

```sh
bun install
```

### Development

```sh
bun dev
```

The app will be available at `http://localhost:5173`

### Production Build

```sh
bun run build
```
